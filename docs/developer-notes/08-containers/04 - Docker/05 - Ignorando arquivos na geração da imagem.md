### dockerignore

Caso necessario fazer o build de uma imagem e tenha arquivos que não são necessarios, pois serão gerados novamentos, tipo a pasta de modulos do npm utilize o arquivo **.dockerignore** para ignorá-los  


- Crie o arquivo .dockerignore
- Adicione as pastas ou arquivos que deseja ignorar
